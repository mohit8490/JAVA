## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/YOUR_USERNAME/Java.git
   ```

2. Open the project in your preferred IDE (VS Code, IntelliJ IDEA, or Eclipse).

3. Compile the Java file:

   ```bash
   javac FileName.java
   ```

4. Run the program:

   ```bash
   java FileName
   ```
